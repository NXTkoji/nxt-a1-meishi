Filter chips
Active filters shown as dismissible chips. Tap "+ Filter" to add more. Available filters: Year, Month, Date, Occasion, and a quick "Not yet exported to any destination".
Text search
Searches across all text fields: name (all languages), company, title, department, email, phone, address, notes.
Select all / Deselect all
Applies only to current filtered results — not to all cards in the database. Individual rows can then be toggled on or off.
Sync badges
Subtle badges show which destinations this card has already been exported to. Informational only — doesn't block re-export.
