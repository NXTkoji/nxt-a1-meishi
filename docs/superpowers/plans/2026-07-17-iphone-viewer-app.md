# iPhone View/Edit App Implementation Plan

> **SUPERSEDED (2026-07-18):** This plan is on hold. After building Task 3 (Google Drive snapshot export), discovered that a custom SwiftUI app sideloaded without the $99/year Apple Developer Program requires re-signing in Xcode every 7 days — an ongoing cost/hassle not previously surfaced. Decided instead to auto-push every card to Google Contacts (`app/services/google_contacts.py`, currently manual-trigger-only) and use the native iOS Contacts app / Google Contacts app as the phone-side viewer. Google Contacts is explicitly a **read-mostly viewer**: edits made there do not flow back to this app's database, and some fields won't have a home there — a "Met As" custom field (People API `userDefined`) will be added to close some of that gap. Tasks 1–2 (PATCH notes endpoint, image route auth) and Task 3 (Drive snapshot export service) are already committed on `feat/iphone-viewer-app` and are harmless to leave as-is, just unused for now. Do not resume Tasks 4+ without first re-confirming this decision still holds.

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship a native SwiftUI iPhone app that browses, searches, views, and edits the contacts already captured by NXT-A1 — synced through a dedicated Google Drive folder rather than a live connection to the backend.

**Architecture:** A new `ios/` folder in this repo holds an Xcode project (generated by `xcodegen` from a checked-in `project.yml`) containing a SwiftUI app that authenticates with Google Sign-In and reads/writes data in a private Drive folder via the Drive API v3 — it never talks to the FastAPI backend directly. The backend (already running locally on Golden-Zeus whenever the laptop is open, via its existing LaunchAgent) runs a periodic background loop (every 60s) that polls a Drive `outbox/` subfolder for edit requests the iPhone app has submitted, applies them to the live SQLite database, and exports a full JSON snapshot of the database plus card images back to Drive. **This makes sync eventually consistent, by design:** the iPhone shows data as of Golden-Zeus's last export (within about a minute of the last time it was open); edits made on the iPhone apply within about a minute of the next time Golden-Zeus is open and the backend is running. Two small backend gaps (found while writing the original version of this plan) were closed first: a missing endpoint to edit `Person.notes`, and an unauthenticated image route — both still useful (the notes endpoint's logic is mirrored by the outbox processor; the image auth is harmless defense-in-depth) even though the backend is no longer internet-facing.

**Tech Stack:** Python/FastAPI/SQLAlchemy (existing backend, `venv/bin/python3`), `httpx` for Drive API calls (matching the existing `google_contacts.py` OAuth pattern), Swift 5.9 / SwiftUI / `@Observable` (iOS 17+ deployment target), `xcodegen` for project generation, [GoogleSignIn-iOS](https://github.com/google/GoogleSignIn-iOS) SDK (added as an SPM dependency) for on-device Google auth, Google Drive API v3 (called directly over REST from both Python and Swift — no dependency on the Google Drive desktop sync client).

**Deviations, in order discovered:**
1. The original spec assumed `PATCH /api/v2/persons/{id}` already existed for editing notes. It didn't — Task 1 added it. *(done)*
2. The original spec didn't anticipate that `GET /api/v2/images/{path}` had no auth check, and that securing it with a header-only check would break the web app's plain `<img src>` tags. Task 2 added a query-param key alternative and updated the 4 web pages that load images. *(done)*
3. **Architecture change (this revision):** the plan originally exposed the backend via Cloudflare Tunnel with a public HTTPS hostname (former Tasks 3–4). That was abandoned after discovering, in order: (a) Cloudflare subdomain-zone delegation is Enterprise-plan only, so it couldn't avoid touching `nxta.co`'s root DNS (which carries live company email) without buying a second domain; (b) `nxta.co` is on GoDaddy nameservers with live Google Workspace MX records, making a full migration risky; (c) Golden-Zeus is a laptop that's closed most of the time, so a live tunnel would be unreachable whenever the lid is shut regardless of DNS provider. Google Drive export/outbox sync sidesteps all three: no DNS changes, no paid plan, and reads/writes both work opportunistically whenever Golden-Zeus happens to be open, with no requirement that it be open *right now*.

```python
production data — everything runs against a throwaway temp SQLite file.

import pytest
from fastapi.testclient import TestClient
from sqlalchemy.ext.asyncio import async_sessionmaker, create_async_engine

from app.db.models import Base
from app.db.session import get_db

def client_with_test_db(tmp_path):

    engine = create_async_engine(
        f"sqlite+aiosqlite:///{db_path}",
        connect_args={"check_same_thread": False},
    )
    session_maker = async_sessionmaker(bind=engine, expire_on_commit=False)

    async def _create_schema():
        async with engine.begin() as conn:
            await conn.run_sync(Base.metadata.create_all)

        async with session_maker() as session:
            try:
                yield session
                await session.commit()
            except Exception:
```

```python
from fastapi import Header, HTTPException, Query

from app.config import settings


async def verify_api_key(authorization: str = Header(default="")) -> None:
    # If no api_key is configured, the app is running locally — skip auth.
    if not settings.api_key:
        return
    if authorization != f"Bearer {settings.api_key}":
        raise HTTPException(status_code=401, detail="Invalid API key")


async def verify_api_key_flexible(
    authorization: str = Header(default=""),
    key: str = Query(default=""),
) -> None:
    """Like verify_api_key, but also accepts the key as a `?key=` query
    param. Needed for the image route, which is loaded via plain <img src>
    tags in the web frontend and can't set an Authorization header."""
    if not settings.api_key:
        return
    if authorization == f"Bearer {settings.api_key}":
        return
    if key == settings.api_key:
        return
    raise HTTPException(status_code=401, detail="Invalid API key")
```

```python
"""Export the full database as a JSON snapshot to a dedicated Google Drive
folder, so the iPhone app can read it without ever talking to this backend
directly. Mirrors the OAuth pattern already used in google_contacts.py.
"""
from __future__ import annotations

import json
import logging

import httpx
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession
from sqlalchemy.orm import selectinload

from app.config import settings
from app.db.models import Card, Person
from app.schemas.api import CardOut, PersonOut
from app.routers.v2.persons import _load_person_out

logger = logging.getLogger(__name__)

TOKEN_URL = "https://oauth2.googleapis.com/token"
DRIVE_UPLOAD_URL = "https://www.googleapis.com/upload/drive/v3/files"
DRIVE_FILES_URL = "https://www.googleapis.com/drive/v3/files"

_snapshot_file_id_cache: str | None = None


async def _get_access_token() -> str:
    async with httpx.AsyncClient() as client:
        resp = await client.post(TOKEN_URL, data={
            "client_id": settings.google_client_id,
            "client_secret": settings.google_client_secret,
            "refresh_token": settings.google_refresh_token,
            "grant_type": "refresh_token",
        })
        resp.raise_for_status()
        return resp.json()["access_token"]


async def _build_snapshot(db: AsyncSession) -> dict:
    persons = (await db.scalars(
        select(Person).where(Person.deleted_at.is_(None))
    )).all()
    person_outs = [await _load_person_out(db, p) for p in persons]

    cards = (await db.scalars(
        select(Card)
        .where(Card.deleted_at.is_(None))
        .options(selectinload(Card.sides), selectinload(Card.my_company_links))
    )).all()
    card_outs = []
    for card in cards:
        person_ext_id = await db.scalar(
            select(Person.external_id).where(Person.id == card.person_id)
        )
        out = CardOut.model_validate(card)
        out.person_external_id = person_ext_id
        out.my_company_ids = [link.my_company_id for link in card.my_company_links]
        card_outs.append(out)

    return {
        "persons": [json.loads(p.model_dump_json()) for p in person_outs],
        "cards": [json.loads(c.model_dump_json()) for c in card_outs],
    }


async def _find_existing_file_id(access_token: str, name: str) -> str | None:
    global _snapshot_file_id_cache
    if name == "snapshot.json" and _snapshot_file_id_cache:
        return _snapshot_file_id_cache
    async with httpx.AsyncClient() as client:
        resp = await client.get(
            DRIVE_FILES_URL,
            headers={"Authorization": f"Bearer {access_token}"},
            params={
                "q": f"name = '{name}' and '{settings.google_drive_folder_id}' in parents and trashed = false",
                "fields": "files(id)",
            },
        )
        resp.raise_for_status()
        files = resp.json().get("files", [])
        return files[0]["id"] if files else None


async def _upload_json(access_token: str, name: str, payload: dict) -> str:
    body = json.dumps(payload, ensure_ascii=False, indent=2).encode("utf-8")
    existing_id = await _find_existing_file_id(access_token, name)

    async with httpx.AsyncClient() as client:
        if existing_id:
            resp = await client.patch(
                f"{DRIVE_UPLOAD_URL}/{existing_id}?uploadType=media",
                headers={"Authorization": f"Bearer {access_token}", "Content-Type": "application/json"},
                content=body,
            )
        else:
            metadata = {"name": name, "parents": [settings.google_drive_folder_id]}
            resp = await client.post(
                f"{DRIVE_UPLOAD_URL}?uploadType=multipart",
                headers={"Authorization": f"Bearer {access_token}"},
                files={
                    "metadata": ("metadata", json.dumps(metadata), "application/json"),
                    "file": ("file", body, "application/json"),
                },
            )
        resp.raise_for_status()
        file_id = resp.json()["id"]
        if name == "snapshot.json":
            global _snapshot_file_id_cache
            _snapshot_file_id_cache = file_id
        return file_id


async def export_snapshot(db: AsyncSession) -> None:
    """Build a full snapshot of persons+cards and upload it to Drive as
    snapshot.json, creating it on first run and updating it in place after."""
    access_token = await _get_access_token()
    snapshot = await _build_snapshot(db)
    await _upload_json(access_token, "snapshot.json", snapshot)
    logger.info("Exported snapshot: %d persons, %d cards", len(snapshot["persons"]), len(snapshot["cards"]))
```

```python
"""Poll the Drive outbox/ folder for edit requests submitted by the iPhone
app, apply each to the live database, and move processed files out of the
way. Runs on a periodic background loop from app/main.py's lifespan."""
from __future__ import annotations

import json
import logging

import httpx
from sqlalchemy import select
from sqlalchemy.ext.asyncio import AsyncSession

from app.config import settings
from app.db.models import ContactDetail, OrganizationName, Person, PersonName, PositionDetail
from app.services.drive_sync import _get_access_token, DRIVE_FILES_URL

logger = logging.getLogger(__name__)


async def apply_edit(db: AsyncSession, edit: dict) -> None:
    op = edit.get("op")
    person_ext_id = edit.get("person_external_id")

    if op == "update_person_notes":
        person = await db.scalar(select(Person).where(Person.external_id == person_ext_id))
        if person:
            person.notes = edit["notes"]
        return

    if op == "delete_person":
        person = await db.scalar(select(Person).where(Person.external_id == person_ext_id))
        if person:
            await db.delete(person)
        return

    if op == "update_person_name":
        name = await db.get(PersonName, edit["name_id"])
        if name:
            for field in ("full_name", "family_name", "given_name", "honorific"):
                if edit.get(field) is not None:
                    setattr(name, field, edit[field])
            name.source = "manual"
        return

    if op == "update_org_name":
        org_name = await db.get(OrganizationName, edit["org_name_id"])
        if org_name and edit.get("name") is not None:
            org_name.name = edit["name"]
        return

    if op == "update_contact_detail":
        detail = await db.get(ContactDetail, edit["detail_id"])
        if detail:
            for field in ("value", "label", "detail_type", "country_code"):
                if edit.get(field) is not None:
                    setattr(detail, field, edit[field])
        return

    if op == "add_contact_detail":
        person = await db.scalar(select(Person).where(Person.external_id == person_ext_id))
        if person:
            db.add(ContactDetail(
                person_id=person.id,
                value=edit["value"],
                detail_type=edit["detail_type"],
                label=edit.get("label"),
                is_primary=False,
            ))
        return

    if op == "delete_contact_detail":
        detail = await db.get(ContactDetail, edit["detail_id"])
        if detail:
            await db.delete(detail)
        return

    if op == "update_position_detail":
        detail = await db.get(PositionDetail, edit["detail_id"])
        if detail:
            for field in ("title", "department"):
                if edit.get(field) is not None:
                    setattr(detail, field, edit[field])
        return

    raise ValueError(f"Unknown outbox op: {op}")


async def _list_outbox_files(access_token: str) -> list[dict]:
    async with httpx.AsyncClient() as client:
        resp = await client.get(
            DRIVE_FILES_URL,
            headers={"Authorization": f"Bearer {access_token}"},
            params={
                "q": f"'{settings.google_drive_folder_id}' in parents and name contains 'outbox-' and trashed = false",
                "fields": "files(id, name)",
            },
        )
        resp.raise_for_status()
        return resp.json().get("files", [])


async def poll_and_apply_outbox(db: AsyncSession) -> int:
    """Check Drive for pending edit-request files, apply and delete each.
    Returns the number of edits applied."""
    access_token = await _get_access_token()
    files = await _list_outbox_files(access_token)
    applied = 0

    async with httpx.AsyncClient() as client:
        for f in files:
            resp = await client.get(
                f"{DRIVE_FILES_URL}/{f['id']}?alt=media",
                headers={"Authorization": f"Bearer {access_token}"},
            )
            resp.raise_for_status()
            try:
                edit = resp.json()
                await apply_edit(db, edit)
                applied += 1
            except Exception:
                logger.exception("Failed to apply outbox edit %s", f["name"])
                continue
            await client.delete(
                f"{DRIVE_FILES_URL}/{f['id']}",
                headers={"Authorization": f"Bearer {access_token}"},
            )

    if applied:
        await db.commit()
    return applied
```

- [ ] **Step 4b: Implement image upload to Drive**

Card images (`~/.nxt-a1/images/{card_ext_id}/{side_order}.jpg` on disk, referenced as `"{card_ext_id}/{side_order}.jpg"` in `CardSideOut.image_path`) need to reach Drive too, so the iPhone app can show them without ever contacting Golden-Zeus directly. import json as _json
```python
from pathlib import Path

_IMAGE_MANIFEST_PATH = settings.data_dir / "drive_synced_images.json"


def _flatten_image_path(image_path: str) -> str:
    return image_path.replace("/", "__")


def _load_image_manifest() -> dict:
    if _IMAGE_MANIFEST_PATH.exists():
        return _json.loads(_IMAGE_MANIFEST_PATH.read_text())
    return {}


def _save_image_manifest(manifest: dict) -> None:
    _IMAGE_MANIFEST_PATH.write_text(_json.dumps(manifest))


async def upload_pending_images(db: AsyncSession) -> int:
    """Upload any card image not yet in the local sync manifest to the
    Drive images/ subfolder. Returns the count of newly-uploaded images."""
    cards = (await db.scalars(
        select(Card).where(Card.deleted_at.is_(None)).options(selectinload(Card.sides))
    )).all()

    manifest = _load_image_manifest()
    uploaded = 0
    access_token: str | None = None

    for card in cards:
        for side in card.sides:
            if side.image_path in manifest:
                continue
            local_path = settings.images_path / side.image_path
            if not local_path.exists():
                continue
            if access_token is None:
                access_token = await _get_access_token()
            flat_name = _flatten_image_path(side.image_path)
            metadata = {"name": flat_name, "parents": [settings.google_drive_images_folder_id]}
            async with httpx.AsyncClient() as client:
                resp = await client.post(
                    f"{DRIVE_UPLOAD_URL}?uploadType=multipart",
                    headers={"Authorization": f"Bearer {access_token}"},
                    files={
                        "metadata": ("metadata", json.dumps(metadata), "application/json"),
                        "file": ("file", local_path.read_bytes(), "image/jpeg"),
                    },
                )
                resp.raise_for_status()
                manifest[side.image_path] = resp.json()["id"]
                uploaded += 1

    if uploaded:
        _save_image_manifest(manifest)
    return uploaded
```

- [ ] **Step 5: Wire a periodic loop into app startup**

In `app/main.py`, find the existing lifespan context manager (or add one if the app currently starts with `app = FastAPI()` directly — check first) and add a background task that runs every 60 seconds for as long as the process is alive:
```python
import asyncio
import contextlib

from app.db.session import async_session
from app.services import drive_outbox, drive_sync

SYNC_POLL_INTERVAL_SECONDS = 60


async def _sync_loop():
    while True:
        try:
            async with async_session() as db:
                await drive_outbox.poll_and_apply_outbox(db)
                # Always re-export and re-check images, not just when the
                # outbox had something — edits made via the existing local
                # web app (or a freshly-scanned card) don't go through the
                # outbox at all, but still need to reach Drive.
                await drive_sync.upload_pending_images(db)
                await drive_sync.export_snapshot(db)
        except Exception:
            logging.getLogger(__name__).exception("Sync loop iteration failed")
        await asyncio.sleep(SYNC_POLL_INTERVAL_SECONDS)


@contextlib.asynccontextmanager
async def lifespan(app: FastAPI):
    task = asyncio.create_task(_sync_loop())
    yield
    task.cancel()
    with contextlib.suppress(asyncio.CancelledError):
        await task
```
Pass `lifespan=lifespan` to the `FastAPI(...)` constructor. If a lifespan already exists (check for `@app.on_event` or an existing `lifespan=` argument before adding a second one — merge into the existing one rather than adding a duplicate).

Note: this exports on a 60-second cadence rather than synchronously after every single mutation. That's a deliberate simplification — it avoids threading a background-export call through every existing mutation endpoint (person names, positions, contact details, cards each live in different router files), and "eventually consistent within about a minute, only while Golden-Zeus is open anyway" is well within what this design already promises.

```swift
import Foundation
import GoogleSignIn
import Observation

@Observable
final class GoogleAuthManager {
    static let shared = GoogleAuthManager()

    var currentUser: GIDGoogleUser?
    var isSignedIn: Bool { currentUser != nil }

    private init() {}

    func restorePreviousSignIn() async {
        currentUser = try? await GIDSignIn.sharedInstance.restorePreviousSignIn()
    }

    @MainActor
    func signIn(presenting viewController: UIViewController) async throws {
        let result = try await GIDSignIn.sharedInstance.signIn(
            withPresenting: viewController,
            hint: nil,
            additionalScopes: [DriveSyncConfig.driveScope]
        )
        currentUser = result.user
    }

    func signOut() {
        GIDSignIn.sharedInstance.signOut()
        currentUser = nil
    }

    /// Fetches a fresh access token, refreshing under the hood if the
    /// cached one is near expiry. Every DriveSyncClient call goes through
    /// this rather than caching a token itself.
    func accessToken() async throws -> String {
        guard let user = currentUser else { throw SyncError.notSignedIn }
        return try await withCheckedThrowingContinuation { continuation in
            user.refreshTokensIfNeeded { user, error in
                if let error {
                    continuation.resume(throwing: error)
                } else if let token = user?.accessToken.tokenString {
                    continuation.resume(returning: token)
                } else {
                    continuation.resume(throwing: SyncError.notSignedIn)
                }
            }
        }
    }
}
```

```swift
import Foundation
import XCTest

final class MockURLProtocol: URLProtocol {
    static var requestHandler: ((URLRequest) throws -> (HTTPURLResponse, Data))?

    override class func canInit(with request: URLRequest) -> Bool { true }
    override class func canonicalRequest(for request: URLRequest) -> URLRequest { request }

    override func startLoading() {
        guard let handler = Self.requestHandler else {
            XCTFail("No request handler set")
            return
        }
        do {
            let (response, data) = try handler(request)
            client?.urlProtocol(self, didReceive: response, cacheStoragePolicy: .notAllowed)
            client?.urlProtocol(self, didLoad: data)
            client?.urlProtocolDidFinishLoading(self)
        } catch {
            client?.urlProtocol(self, didFailWithError: error)
        }
    }

    override func stopLoading() {}

    static func makeTestSession() -> URLSession {
        let config = URLSessionConfiguration.ephemeral
        config.protocolClasses = [MockURLProtocol.self]
        return URLSession(configuration: config)
    }
}
```

```swift
import Foundation

struct DriveFile: Decodable {
    let id: String
    let name: String
}

final class DriveSyncClient {
    static let shared = DriveSyncClient()

    static let filesURL = "https://www.googleapis.com/drive/v3/files"
    static let uploadURL = "https://www.googleapis.com/upload/drive/v3/files"

    private let tokenProvider: () async throws -> String
    private let session: URLSession
    let decoder: JSONDecoder
    let encoder: JSONEncoder

    init(
        tokenProvider: @escaping () async throws -> String = { try await GoogleAuthManager.shared.accessToken() },
        session: URLSession = .shared
    ) {
        self.tokenProvider = tokenProvider
        self.session = session
        self.decoder = JSONDecoder()
        self.decoder.keyDecodingStrategy = .convertFromSnakeCase
        self.encoder = JSONEncoder()
        self.encoder.keyEncodingStrategy = .convertToSnakeCase
    }

    private func authorizedRequest(url: URL, method: String = "GET") async throws -> URLRequest {
        let token = try await tokenProvider()
        var request = URLRequest(url: url)
        request.httpMethod = method
        request.setValue("Bearer \(token)", forHTTPHeaderField: "Authorization")
        return request
    }

    private func perform(_ request: URLRequest) async throws -> Data {
        let data: Data
        let response: URLResponse
        do {
            (data, response) = try await session.data(for: request)
        } catch {
            throw SyncError.network
        }
        guard let http = response as? HTTPURLResponse else { throw SyncError.network }
        switch http.statusCode {
        case 200..<300:
            return data
        case 404:
            throw SyncError.notFound
        default:
            let message = String(data: data, encoding: .utf8) ?? ""
            throw SyncError.drive(status: http.statusCode, message: message)
        }
    }

    func listFiles(inFolder folderID: String, nameContains: String? = nil) async throws -> [DriveFile] {
        var q = "'\(folderID)' in parents and trashed = false"
        if let nameContains {
            q += " and name contains '\(nameContains)'"
        }
        var components = URLComponents(string: Self.filesURL)!
        components.queryItems = [
            URLQueryItem(name: "q", value: q),
            URLQueryItem(name: "fields", value: "files(id, name)"),
        ]
        let request = try await authorizedRequest(url: components.url!)
        let data = try await perform(request)
        struct FileList: Decodable { let files: [DriveFile] }
        do {
            return try decoder.decode(FileList.self, from: data).files
        } catch {
            throw SyncError.decoding
        }
    }

    func downloadFile(id: String) async throws -> Data {
        var components = URLComponents(string: "\(Self.filesURL)/\(id)")!
        components.queryItems = [URLQueryItem(name: "alt", value: "media")]
        let request = try await authorizedRequest(url: components.url!)
        return try await perform(request)
    }

    func deleteFile(id: String) async throws {
        let url = URL(string: "\(Self.filesURL)/\(id)")!
        let request = try await authorizedRequest(url: url, method: "DELETE")
        _ = try await perform(request)
    }

    /// Creates a new file, or overwrites it in place if `existingFileID` is given.
    /// Returns the resulting file's ID.
    func uploadJSON(name: String, parentFolderID: String, body: Data, existingFileID: String? = nil) async throws -> String {
        let token = try await tokenProvider()
        var request: URLRequest

        if let existingFileID {
            request = URLRequest(url: URL(string: "\(Self.uploadURL)/\(existingFileID)?uploadType=media")!)
            request.httpMethod = "PATCH"
            request.setValue("Bearer \(token)", forHTTPHeaderField: "Authorization")
            request.setValue("application/json", forHTTPHeaderField: "Content-Type")
            request.httpBody = body
        } else {
            let boundary = UUID().uuidString
            request = URLRequest(url: URL(string: "\(Self.uploadURL)?uploadType=multipart")!)
            request.httpMethod = "POST"
            request.setValue("Bearer \(token)", forHTTPHeaderField: "Authorization")
            request.setValue("multipart/related; boundary=\(boundary)", forHTTPHeaderField: "Content-Type")

            let metadata = try JSONSerialization.data(withJSONObject: ["name": name, "parents": [parentFolderID]])
            var multipart = Data()
            multipart.append("--\(boundary)\r\nContent-Type: application/json\r\n\r\n".data(using: .utf8)!)
            multipart.append(metadata)
            multipart.append("\r\n--\(boundary)\r\nContent-Type: application/json\r\n\r\n".data(using: .utf8)!)
            multipart.append(body)
            multipart.append("\r\n--\(boundary)--".data(using: .utf8)!)
            request.httpBody = multipart
        }

        let data = try await perform(request)
        struct CreatedFile: Decodable { let id: String }
        do {
            return try JSONDecoder().decode(CreatedFile.self, from: data).id
        } catch {
            throw SyncError.decoding
        }
    }
}
```

```swift
import Foundation
import Observation

@Observable
final class SyncStore {
    static let shared = SyncStore()

    var snapshot: ExportSnapshot?
    var lastSyncedAt: Date?
    var isSyncing = false
    var errorMessage: String?

    private let client: DriveSyncClient
    private let cacheURL: URL

    init(client: DriveSyncClient = .shared) {
        self.client = client
        let docs = FileManager.default.urls(for: .documentDirectory, in: .userDomainMask)[0]
        self.cacheURL = docs.appendingPathComponent("snapshot-cache.json")
    }

    /// Loads the last-synced snapshot from disk so the app shows something
    /// immediately at launch, even fully offline, before/without a fresh sync.
    func loadCachedSnapshot() {
        guard let data = try? Data(contentsOf: cacheURL) else { return }
        let decoder = JSONDecoder()
        decoder.keyDecodingStrategy = .convertFromSnakeCase
        snapshot = try? decoder.decode(ExportSnapshot.self, from: data)
    }

    func sync() async {
        isSyncing = true
        errorMessage = nil
        do {
            let fresh = try await client.downloadSnapshot()
            snapshot = fresh
            lastSyncedAt = Date()
            let encoder = JSONEncoder()
            encoder.keyEncodingStrategy = .convertToSnakeCase
            if let data = try? encoder.encode(fresh) {
                try? data.write(to: cacheURL)
            }
        } catch {
            errorMessage = (error as? SyncError)?.errorDescription ?? error.localizedDescription
        }
        isSyncing = false
    }

    /// Submits an edit to the Drive outbox. Does not update `snapshot`
    /// locally — see the design note above this task.
    func submitEdit(_ request: OutboxEditRequest) async throws {
        try await client.submitEdit(request)
    }
}
```

```swift
import SwiftUI

@Observable
final class RemoteImageLoader {
    var image: UIImage?
    var isLoading = false
    var errorMessage: String?

    private static let cacheDir: URL = {
        let dir = FileManager.default.urls(for: .cachesDirectory, in: .userDomainMask)[0]
            .appendingPathComponent("card-images", isDirectory: true)
        try? FileManager.default.createDirectory(at: dir, withIntermediateDirectories: true)
        return dir
    }()

    func load(path: String, client: DriveSyncClient = .shared) async {
        let cacheURL = Self.cacheDir.appendingPathComponent(path.replacingOccurrences(of: "/", with: "__"))

        if let cached = try? Data(contentsOf: cacheURL), let uiImage = UIImage(data: cached) {
            image = uiImage
            return
        }

        isLoading = true
        errorMessage = nil
        do {
            let data = try await client.downloadImage(path: path)
            try? data.write(to: cacheURL)
            image = UIImage(data: data)
        } catch {
            errorMessage = (error as? SyncError)?.errorDescription ?? error.localizedDescription
        }
        isLoading = false
    }
}

                    .foregroundStyle(.secondary)
            } else {
                Color.clear
            }
        }
```

```swift
@Observable
final class PersonDetailViewModel {
    let personExternalId: String

    private let syncStore: SyncStore

    init(personExternalId: String, syncStore: SyncStore = .shared) {
        self.personExternalId = personExternalId
        self.syncStore = syncStore

            $0.fullName = body.fullName
            $0.familyName = body.familyName
            $0.givenName = body.givenName
            $0.honorific = body.honorific
        }
    }

        try await submit(op: "update_org_name") {
            $0.orgNameId = orgNameId
            $0.name = body.name
        }
    }

    func saveContactDetail(detailId: Int, body: ContactDetailUpdateBody) async throws {
        try await submit(op: "update_contact_detail") {
            $0.detailId = detailId
            $0.value = body.value
            $0.label = body.label
            $0.detailType = body.detailType
            $0.countryCode = body.countryCode
        }
    }

    func addContactDetail(body: ContactDetailUpdateBody) async throws {
        try await submit(op: "add_contact_detail") {
            $0.value = body.value
            $0.label = body.label
            $0.detailType = body.detailType
            $0.countryCode = body.countryCode
        }
    }

    func deleteContactDetail(detailId: Int) async throws {
        try await submit(op: "delete_contact_detail") { $0.detailId = detailId }
    }
```
